# 技术栈

## 核心语言与运行时
- **Python**: 项目的主要编程语言。
- **Ray (v2.53.0)**: 用于分布式计算、数据处理和模型推理调度的核心框架。

## 推理引擎与计算
- **vLLM**: 用于大语言模型（LLM）的高性能批量推理引擎。
- **CUDA**: 所有的推理任务和高性能算子均依赖于 NVIDIA CUDA 进行硬件加速。

## 数据处理
- **Ray Data**: 用于加载、转换和分发大规模多模态数据集。

## 开发工具
- **Gemini CLI (Conductor)**: 用于项目生命周期管理、任务拆解与 AI 辅助开发。
- **Git**: 版本控制系统，要求提交信息使用中文。
